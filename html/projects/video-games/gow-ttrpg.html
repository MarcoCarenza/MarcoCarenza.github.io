OVERVIEW

A TableTop Role Playing Game (TTRPG) inspired by the Gears of War franchise created to emulate the video games in RPG format. The game is based on a 2d8 system with exploding dice. It allows for controlled but varied results when rolling for tasks.

The focus is on combat, which is core to the games. Progression occurs through building up skills and gaining Perks, rather than gaining Levels. Classes exist as specializations, granting characters abilities and differing starting equipment.

I created a campaign, a starter adventure, pre-made characters, as well as the core rule book for this game.

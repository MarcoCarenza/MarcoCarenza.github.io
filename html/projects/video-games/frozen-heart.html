OVERVIEW

A procedurally generated horror game set in the year 30XX, you are tasked to drill for water on a frozen planet for Aquarii Inc. However, once you arrive you notice that you’re not the only one at the facility, as there is a monster trying to stop you! You need to explore the ever-changing facility while making sure that the drill keeps running.

For this project I was the lead designer, lead programmer, co-level designer, and producer. I created most of the level segments, and designed the random generation for the drilling facility, responsible for the monster’s AI behavior and playmovement mechanics.

For this project we were running a SCRUM-like framework. I organized weekly sprints where we would run retrospectives, each week determining how the project would evolve and seeing how we could improve.

We had a Kanban style project backlog on Trello, where we would list out all tasks to do, and move them around to know what was being worked on and by who.
